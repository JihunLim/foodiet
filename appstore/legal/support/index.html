자주 묻는 질문

1. 사진을 찍었는데 분석이 계속 "분석중" 이에요.

  • Wi-Fi / LTE 연결을 확인해줘. 오프라인 상태면 큐에 저장돼 있다가 온라인 복귀 후 자동 업로드 돼.
  • 1–2분 기다려도 바뀌지 않으면 기록 상세에서 수정 → 다시 저장으로 재시도.
  • 문제가 계속되면 이메일로 스크린샷과 함께 보내줘.

2. AI 가 메뉴를 잘못 맞혔어요.

기록 상세 우상단 메뉴 → 수정 에서 끼니·시간·공유 인원·메모를 바꿀 수 있어. 품목 단위 편집은 로드맵에 있어 (다음 마이너 버전 예정).

3. 홈 위젯이 업데이트가 안 돼요.

  • iOS 홈 화면 > 위젯 편집에서 foodiet 위젯을 다시 추가해봐.
  • 위젯은 보통 1시간에 한 번 새로고침 돼. 즉시 반영하려면 앱을 한 번 열면 돼.

4. 계정을 삭제하고 싶어요.

앱 내 설정 > 계정 삭제 또는 이메일 요청으로 전체 데이터 삭제 가능. 30일 내 완전 삭제 처리.

5. 광고가 거슬려요.

인사이트 탭에 네이티브 광고 1개가 삽입돼 있어. 광고 없는 프리미엄 버전은 향후 업데이트에서 예정.

6. Apple / Google / Kakao 로그인이 안 돼요.

  • iOS 설정 > Apple ID > 로그인 및 보안에서 2단계 인증이 켜져 있는지 확인.
  • 같은 이메일로 다른 제공자 계정이 이미 있는 경우 첫 가입한 제공자로 로그인 해야 해.
  • 해결이 안 되면 이메일로 사용 중인 이메일 주소와 가입 시도한 제공자를 알려줘.

7. 개인정보가 안전한가요?

사진은 Supabase 비공개 Storage 에만 저장되고, 외부에서 URL 로 직접 접근할 수 없어. 자세한 처리 정책은 개인정보처리방침에서 확인해.
