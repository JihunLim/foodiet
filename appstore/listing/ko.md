## Description (설명) — 최대 4,000자

**foodiet — 사진 한 장으로 시작하는 AI 식단 파트너**

식단 기록, 아직도 메뉴 검색 → 양 입력 → 끼니 선택으로 30초씩 쓰고 있다면 멈춰도 돼.
foodiet 은 사진 한 장만 찍으면 AI 푸디가 알아서 분석해.
메뉴 이름, 양, 칼로리, 탄단지 비율, 아침·점심·저녁 끼니 구분까지 전부 자동.

■ 핵심 기능

• 사진 한 장으로 기록
  — 찍기만 하면 AI가 품목별로 칼로리와 양을 추정해.
  — 야식이면 야식, 간식이면 간식. 시간과 사진 기반으로 끼니도 자동 분류.

• 탄단지·영양 분해
  — 탄수화물 · 단백질 · 지방 그램 수와 비율을 한눈에.
  — 신뢰도 점수로 AI가 얼마나 확신하는지도 알려줘.

• 1인분 나눔 계산
  — 친구랑 나눠먹었다면 "n명"만 지정해. 전체 칼로리 ÷ 인원으로 내 몫만 기록돼.

• 푸디의 매일 코치
  — 목표 칼로리 대비 오늘 먹은 식단을 푸디가 짧고 따뜻한 말투로 코치해.
  — 과식한 날도, 잘 먹은 날도 푸디는 몰아세우지 않아.

• 홈 위젯
  — 홈 화면 위젯으로 "오늘 남은 kcal" 를 한눈에 확인.
  — 위젯 탭하면 카메라로 바로 진입.

• 하루 식단 공유
  — PT 친구·코치에게 보낼 때, 오늘 먹은 사진들을 카드 한 장으로 묶어 공유.
  — 과거 날짜도 기록탭에서 일자별 공유 버튼으로.

• 체중 추적 + 미래 예측
  — 체중을 한 번 찍어두면 그래프 위에 점이 쌓여.
  — 최근 14일 평균 섭취와 TDEE 로 "이대로 가면 기한까지 몇 kg" 을 점선으로 보여줘.
  — 도달 가능성을 🎯 / ⚠️ / 🔴 세 단계로 판정 + 하루 몇 kcal 더 줄이거나
    며칠 늦추면 도달하는지 구체적인 조언까지.

• 인사이트
  — 주간·월간 칼로리 추이, 끼니별 분포, 자주 먹는 품목을 한 화면에.

■ 이런 분께 추천해요

• PT 받으면서 식단을 매일 기록해야 하는 분
• 칼로리 계산은 귀찮지만 감은 알고 싶은 분
• 메뉴 이름·양 입력이 번거로워 식단 앱을 여러 번 포기한 분
• 다이어트 중 혼자보다 누가 옆에서 코치해주면 좋을 분

■ 로그인

• Apple · Google · Kakao 로 간편 로그인.
• 비밀번호 입력 없음. 기기에 저장된 계정으로 1탭.

■ 프라이버시

• 사진은 Supabase Storage 에 서명된 비공개 URL 로 저장돼. 본인만 볼 수 있어.
• 광고 개인화(IDFA) 는 선택 — 거부해도 앱은 전부 돌아가.
• 상세 정책은 설정 > 개인정보처리방침에서 확인할 수 있어.

■ 요금

• 1.0 은 무료. 인사이트 탭에 광고가 들어가.
• 프리미엄(광고 제거 + 고급 코치) 은 향후 버전에서 예정.
