자주 묻는 질문

📸 식단 기록

사진을 찍었는데 분석이 계속 "분석중"이에요.

AI 분석은 서버에 사진이 전달돼야 시작돼. 아래 순서로 확인해봐.

  • Wi-Fi / LTE 연결 상태 확인. 오프라인이면 큐에 저장됐다가 온라인 복귀 후 자동 전송돼.
  • 1–2분 기다려도 안 바뀌면 기록 상세 화면 → 수정 → 다시 저장으로 재시도해봐.
  • 그래도 안 되면 앱을 완전히 껐다 다시 켜봐 (백그라운드 완전 종료).
🌱 계속 문제가 생기면 스크린샷과 함께 이메일로 알려줘. 빠르게 확인할게!
AI가 음식 이름을 잘못 맞혔어요.

기록 상세 화면 우상단 메뉴 → 수정에서 끼니·시간·공유 인원·메모를 직접 바꿀 수 있어.

더 정확한 분석을 위해 음식이 잘 보이도록 위에서 찍거나, 그릇 전체가 프레임에 들어오도록 해봐.

💡 품목 단위(개별 음식) 수정 기능은 다음 업데이트에 추가 예정이야!
끼니(아침/점심/저녁)가 자동으로 잘못 분류됐어요.

foodiet은 사진 촬영 시각과 이미지 내 시각 단서(조명, 플레이팅 등)를 함께 보고 끼니를 추정해.

잘못 분류된 경우 수정에서 끼니 항목을 직접 변경할 수 있어.

여러 명이 함께 먹은 음식의 칼로리를 나누고 싶어요.

기록 화면에서 공유 인원을 설정하면 돼. 예를 들어 2명이 나눠 먹었으면 인원 2로 설정하면 총 칼로리가 자동으로 반으로 나뉘어.

과거 날짜에 기록을 추가하거나 수정할 수 있나요?

히스토리 탭에서 날짜를 선택하면 해당 날의 기록을 확인하고 수정할 수 있어.

새 기록 추가 시에도 날짜를 변경해서 과거 날짜로 저장 가능해.

⚖️ 체중 & 목표

체중 그래프의 예측선(점선)은 어떻게 계산되나요?

최근 14일간의 섭취 칼로리와 내 기초대사량(TDEE)을 비교해서, 지금 속도라면 목표 체중에 언제 도달할지를 추정해.

  • 🎯 On track: 목표 기간 내 달성 예상
  • ⚠️ 조금 빠듯: 약간의 조정이 필요한 상태
  • 🔴 조정 필요: 현재 추이로는 목표 기간 초과 예상

기록 데이터가 쌓일수록 예측이 더 정확해져!

목표 체중이나 목표 날짜를 바꾸고 싶어요.

설정 탭 → 내 목표에서 목표 체중·기간·활동량을 수정할 수 있어.

💡 변경 후 예측선이 자동으로 업데이트돼.

🌸 푸디 코치

푸디 코치 메시지가 오늘 안 와요.

푸디는 당일 기록이 있을 때 코치 메시지를 보내. 기록이 없는 날은 메시지가 생략돼.

  • 알림 설정이 켜져 있는지 확인: iOS 설정 → foodiet → 알림 허용.
  • 앱을 열면 홈 화면에서 오늘의 코치 메시지를 바로 볼 수 있어.
코치 메시지를 하루에 몇 번 받을 수 있나요?

현재 버전에서는 일일 최대 5회까지 코치 메시지가 생성돼. 매 기록마다가 아니라 하루 섭취 흐름을 보고 적절한 타이밍에 보내줘.

🔔 홈 위젯

홈 위젯이 업데이트가 안 돼요.

iOS 위젯은 시스템이 주기적으로 새로고침해서 실시간이 아니야 (보통 1시간 주기).

  • 즉시 반영: 앱을 한 번 열었다 닫으면 위젯 데이터가 업데이트돼.
  • 위젯이 아예 표시가 안 되면 홈 화면 > 위젯 편집에서 foodiet 위젯을 한 번 제거했다가 다시 추가해봐.
위젯을 탭하면 카메라로 바로 이동하나요?

맞아! 위젯을 탭하면 앱이 열리면서 카메라 화면으로 바로 이동해.

📤 PT / 트레이너 공유

트레이너에게 식단을 공유하는 방법이 뭐야?

히스토리 탭에서 공유하고 싶은 날짜를 선택 → 하루 공유 버튼을 탭하면 돼.

음식 사진과 해당 사진의 예상 칼로리(kcal)가 담긴 카드 이미지가 생성되고, 카카오톡·메시지·이메일 등으로 바로 공유할 수 있어.

🔒 공유 카드에는 음식 사진과 칼로리만 포함돼. 체중, 영양소 상세, 코치 메시지는 공유되지 않아.
공유 링크가 있나요? 링크로 트레이너가 볼 수 있나요?

현재 버전에서는 이미지 형태의 공유를 지원해. 링크 기반 공유(트레이너가 브라우저에서 바로 확인)는 v1.1 업데이트에서 추가될 예정이야!

🔐 계정 & 로그인

Apple / Google / Kakao 로그인이 안 돼요.
  • Apple 로그인: iOS 설정 → Apple ID → 로그인 및 보안에서 2단계 인증이 켜져 있는지 확인해.
  • 같은 이메일, 다른 제공자: 처음 가입할 때 사용한 제공자로만 로그인 가능해. 예를 들어 처음 Google로 가입했다면 Apple로 다시 가입하려고 하면 오류가 나.
  • 해결이 안 되면 이메일 주소와 가입 시 사용한 제공자를 이메일로 알려줘.
계정을 삭제하고 싶어요.

앱 내 설정 → 계정 삭제를 통해 직접 삭제할 수 있어. 또는 이메일로 요청하면 처리해줄게.

삭제 요청 후 30일 이내에 모든 개인정보 및 기록이 완전히 삭제돼.

⚠️ 삭제 전 기록 백업(스크린샷 등)을 권장해. 한 번 삭제하면 복구가 불가능해.
비밀번호를 잊어버렸어요.

foodiet은 소셜 로그인(Apple / Google / Kakao)만 지원해. 별도 비밀번호가 없어.

로그인이 안 되면 각 소셜 계정의 비밀번호 재설정을 해당 앱/서비스에서 진행해줘.

기기를 바꿨는데 기록이 사라졌어요.

새 기기에서 기존과 동일한 계정으로 로그인하면 모든 기록이 자동으로 복원돼. 데이터는 서버에 안전하게 저장돼 있어.

🛡️ 개인정보 & 보안

내 식단 사진이 외부에 노출될 수 있나요?

사진은 Supabase 비공개 Storage에만 저장돼. 고유한 서명된 URL 없이는 외부에서 직접 접근이 불가능해.

공유 카드를 생성하지 않는 한 사진은 나만 볼 수 있어.

자세한 데이터 처리 내용은 개인정보처리방침에서 확인해.
내 데이터가 AI 학습에 사용되나요?

foodiet의 AI 분석은 OpenAI의 멀티모달 모델을 사용해. 사용자 사진은 분석 후 OpenAI에 전송되지만, OpenAI의 정책에 따라 API를 통해 전송된 데이터는 모델 학습에 사용되지 않아.

자세한 내용은 개인정보처리방침을 참고해.

💳 광고 & 요금

광고가 거슬려요.

현재 인사이트 탭에 네이티브 광고 1개가 포함돼 있어. 무료로 앱을 운영하기 위한 거야.

광고 없는 프리미엄 구독은 향후 업데이트에서 추가될 예정이야.

foodiet은 무료인가요?

기본 기능(사진 기록, 칼로리 분석, 체중 추적, 코치, 공유)은 모두 무료야.

향후 광고 제거, 추가 분석 기능 등의 프리미엄 플랜이 추가될 예정이야.
